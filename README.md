# Proyecto PHP

## Descripción:
Este proyecto tiene el fin de seguir aprendiendo a diferenciar entre códigos de programación que no necesitan de un compilador para ejecutarse.

## Prerequisitos
* Instalar Visual Studio Code
* Instalar apache2
* Instalar php

## Pasos para su ejecución:
1. Ejecutar Visual Studio Code.
2. Revisar que se tenga instalado apache2 con el comando "apache2 -v".
3. Revisar que se tenga instalado php con el comando "php -v".
4. En caso que no se tenga instalado alguno de los prerequisitos anteriores ejecutar los comandos "sudo apt install apache2" o "sudo apt install php".
5. Buscar la carpeta "/var/www" en Visual Studio Code.
6. Dar permisos con el comando "sudo chown "ingresar su usuario":www-data html -R.
7. Ejecutar comando "sudo chmod 775 html -R".
8. Entrar en la carpeta html con el comando "cd html"
9. Crear carpeta "proyecto_php".
10. Descargar ambos archivos.
11. Copiar archivos en la carpeta "/var/www/html/proyecto_php".
12. Ingresar a la carpeta proyecto_php con el comando "cd proyecto_php".
13. Ingresar a http://localhost/proyecto_php/index.php
